# Todo Claude Mobile

A simple, mobile-friendly todo app built with vanilla JavaScript.

## Features

- Add, complete, and delete tasks
- Filter tasks by all, active, or completed
- Clear all completed tasks
- Persistent storage using localStorage
- Responsive design optimized for mobile devices

## Live Demo

Visit the live app at: [Todo Claude Mobile](https://souvick88.github.io/todo-claude-mobile/)

## Usage

Simply open `index.html` in your browser to start using the app.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API
